Stations expose three track endpoints.

Endpoint references:

- [`getArtistStationTracks`](/openmusic/endpoints/getArtistStationTracks)
- [`getGenreStationTracks`](/openmusic/endpoints/getGenreStationTracks)
- [`getPopularStationTracks`](/openmusic/endpoints/getPopularStationTracks)

## Relationship Endpoints

| Station Type | Endpoint | Response |
| ------------ | -------- | -------- |
| Artist | `GET /stations/artist/{id}/rel/tracks` | [`SongListResponse`](/openmusic/songs/responses#songlistresponse) |
| Genre | `GET /stations/genre/{genre}/rel/tracks` | [`SongListResponse`](/openmusic/songs/responses#songlistresponse) |
| Popular | `GET /stations/popular/rel/tracks` | [`SongListResponse`](/openmusic/songs/responses#songlistresponse) |

## Query Parameters

All station track endpoints support:

| Name | Values | Required |
| ---- | ------ | -------- |
| `lang` | locale string | `false` |
| `include` | `albums`, `artists` | `false` |
| `view` | `minimal` | `false` |
| `view[songs]` | `minimal` | `false` |
| `view[albums]` | `minimal` | `false` |
| `view[artists]` | `minimal` | `false` |
| `limit[albums]` | `1-25` | `false` |
| `limit[artists]` | `1-25` | `false` |

## Responses

- Artist station tracks: [`SongListResponse`](/openmusic/songs/responses#songlistresponse)
- Genre station tracks: [`SongListResponse`](/openmusic/songs/responses#songlistresponse)
- Popular station tracks: [`SongListResponse`](/openmusic/songs/responses#songlistresponse)
- Errors: [`ErrorResponse`](/openmusic/responses#errorresponse)
