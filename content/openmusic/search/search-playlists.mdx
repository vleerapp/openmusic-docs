## Endpoint

`GET` `/search/playlists`

Endpoint reference: [`searchPlaylists`](/openmusic/endpoints/searchPlaylists)

## Required Query Parameter

| Name | Description | Type | Example |
| ---- | ----------- | ---- | ------- |
| `term` | Search term | `string` | `term=workout` |

## Optional Query Parameters

| Name | Values | Required |
| ---- | ------ | -------- |
| `lang` | locale string | `false` |
| `include` | `tracks` | `false` |
| `view` | `minimal` | `false` |
| `view[playlists]` | `minimal` | `false` |
| `view[songs]` | `minimal` | `false` |
| `limit` | `1-25` | `false` |
| `offset` | `0+` | `false` |
| `sort` | `name`, `-name`, `date_added`, `-date_added` | `false` |
| `filter[genres]` | comma-separated genres | `false` |
| `limit[tracks]` | `1-25` | `false` |

## Responses

| Status | Content Type | Response |
| ------ | ------------ | -------- |
| `200 OK` | `application/json` | [`PlaylistPaginatedResponse`](/openmusic/endpoints/searchPlaylists) |
| `400 Bad Request` | `application/json` | [`ErrorResponse`](/openmusic/responses#errorresponse) |
